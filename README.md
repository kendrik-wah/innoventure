# innoventure
Code pertaining to a basic inventory management system within resuscitation trolleys in medical centres.

The software architecture is as such:

Each item has the following criteria:
-------------------------------------
  1) Serial Number
  2) Name
  3) Batch Number
  4) Location
  5) Input Date
  6) Expiry Date

------------------------------
